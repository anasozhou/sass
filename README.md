# sass安装
1. 安装ruby
1. 移除官网sass源   gem sources --remove https://rubygems.org/
1. 添加淘宝源 gem sources -a https://ruby.taobao.org/    
1. fsass -v   查看sass版本

#compass
  * compass安装
    * gem install compass

  * compass常用使用
    * compass -v 参看版本
    * compass create [name] 
   
## cmd命令
* D: (跳转至D盘)
* cd 进入文件夹
* md 创建文件夹
* rd 删除文件夹
* cls 清楚屏幕
* dir 列出文件夹
